# Sales Performance Analysis using Python

This project analyzes sales data from a CSV file and calculates total and product-wise sales.

## Technologies Used
- Python
- CSV
- VS Code
- Excel
- GitHub

## How to Run
1. Keep sales_analysis.py and sales_data.csv in the same folder
2. Run the command:
   python sales_analysis.py

## Learning Outcomes
- File handling in Python
- Using dictionaries for data analysis
- Debugging common errors
